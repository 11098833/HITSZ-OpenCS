## 经济学原理B（经原B）

本课程为必修课，占 3 学分。

分数构成 = 10% 平时分 + 20% 作业 + 35% 期中考试 + 35% 期末考试。

### 关于考试

期中考试比较基础，主要是对主要模型的考察。

期末考试偏难，有一些比较开放的大题，需要综合各个部分学到的模型，想回答得比较完善会很有挑战。
